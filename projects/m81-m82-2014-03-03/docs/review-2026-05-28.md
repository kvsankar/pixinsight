This checkpoint compares the historical reference, dark-calibrated diagnostic, no-dark baseline, and current nonlinear review candidate for the 2014-03-03 M81/M82 data.

## Current Recommendation

Do not finalize either the BXT/NXT v1 branch or the legacy v4 branch. Use the no-dark/no-flats branch as the current reference baseline only, with `m81-m82-20140303-v4-detail-tight-crop.jpg` as the least-bad comparison image.

BXT/NXT v1 makes the artifacts read as colored scratch noise; v4 is less smeared but still not clean enough to call final. NXT-only v2 reduced chroma noise but still showed higher luminance texture than v4. The next useful work is upstream calibration/integration diagnostics rather than more nonlinear polish.
